App Features

No Manual Flipping or Shouting

No more flipping numbers on a chart, writing on a whiteboard, shouting in the parking lot, etc. 100% digital parking lot solution.

Web Based

ParkingLot is a web application. There is no app to install from an app store and it will work on any network connected device (cellular or WiFi).
 
(Native mobile apps are coming soon)

Easy to Use Interface

The app was designed to be clean and simple to use. Even inexperienced volunteers can pick it up in just a few minutes and be able to administer the app like a pro.
The status screen shows which dogs have done which searches. It is handy for the scoreroom as well as the CO to see how the trial is flowing.

Auto Updating

On the client side, the app will automatically refresh and be up to day so the competitor doesn't need to keep hitting the refresh button. Additionally, the app will show the next upcoming dogs so the competitors know to start getting ready.

Trial Check-In

The app includes a check-in feature. When competitors arrive to the trial, they can check-in using the app so a volunteer doesn't have to wander the parking lot looking to see if a competitor arrived or not.

Status Messages

On the competitor screen the admin as the option of displaying a status message. This can be helpful for things like the link to check-in or a message saying "Lunch break until 1:00PM." When set, the message automatically appears and will automatically go away once the message is cleared.
